Detailed Description

An Assignment is an Expression that contains an Identifier and a child Expression. This evaluates to the same Value as the child Expression, after it updates the Value held by the Identifier to that Expression. The following Expression will evaluate to 5: a = 1 + a = 4 The "a" Identifier is set to 4 and the resulting Expression is added with 1. The resulting sum is than saved to "a" and the Assignment will evaluate to this sum (i.e. 5).
