Detailed Description

A ConditionalExpression is an if-statement. It has a condition Expression, a then Expression and an else Expression The result of an if statement is controlled by the evaluation of the condition. If the condition is a non-zero Number this evaluates to the then Expression. Otherwise we evaluate to the else Expression. Note that in the language itself the else Expression is optional, but the AST always has one. If the else is missing in the input file, the parser will insert an Undefined as the else Expression. A ConditionalExpression will convert its condition to a Constant in order to evaluate its result, but the then/else Expressions will be returned as Value.
