Detailed Description

A BinaryExpression is an Expression that contains a LHS (left hand side) operand, a RHS (right hand side) operand and an operator that will be performed on these 2 operands. A basic example is "2 + a", a BinaryExpression that contains a Number as LHS, an Identifier as RHS and adds the 2 values. A BinaryExpressions will convert its operands to Constants before performing the actual operation.
