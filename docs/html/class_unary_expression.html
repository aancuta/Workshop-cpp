Detailed Description

A UnaryExpression is an Expression that contains an operand and an operator that will be performed on this operand. A basic example is "-2", a UnaryExpression that contains a Number as the operand and "-" as the operation. A UnaryExpression will convert its operand to a Constant before performing the actual operation.
