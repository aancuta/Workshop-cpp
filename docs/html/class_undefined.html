Detailed Description

An Undefined Value is a special Value, different than 0, but will evaluate as 0 when we invoke the getAsNumber method. This Value can be produced by reading the Value of an identifier that has not been set or by evaluating an empty Block.

Undefined is an "absorbant" element: x op Undefined == Undefined op x == Undefined
